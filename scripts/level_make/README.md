# Level making
### Step 1 `init level`
1. Run `init level.py`
2. Enter need data
3. Open `games/S2W Adventure` and
    * Put music of game in `sounds` with name `{level_name}-level_sound`
    * Put background image of game in `images` with name `{level_name}-level_background`
    
### Step 2 `make level`
1. Run `edit level.py`
2. Enter need data
3. ### `Make world`
	* «.» Empty Place
	* «-» Horisontal platform
	* «|» Vertical platform
	* «F» Finish platform
	* «D» Death platform
